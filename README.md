# disbit

Приложение для привычек с **добровольной ставкой**: если не выполнил привычку —
ты сам заранее выбираешь, что теряешь. Либо **списание денег** (сумму и получателя
выбираешь сам: благотворительность или создателям), либо **блокировка телефона /
выбранных приложений**. Плюс трекер: дни недели и стрики 🔥.

Идея: помочь тем, кто хочет меняться, но кому не хватает дисциплины и самоконтроля.

## Статус

Ранний прототип. **Ставки сейчас симулируются** — реальные списания и блокировки
требуют нативного приложения и бэкенда с платежами (это следующие этапы).
ИИ-проверку выполнения задач планируется добавить позже.

## Структура

```
frontend/   — веб-прототип (HTML/CSS/JS, данные в localStorage)
backend/    — каркас сервера под будущую базу данных (Express)
```

## Запуск фронтенда

Просто открой `frontend/index.html` в браузере. Данные хранятся локально в браузере.

## Запуск бэкенда

См. [`backend/README.md`](backend/README.md).

## Что уже сделано

- Экран «Сегодня»: неделя, кольцо прогресса, карточки привычек со стриком и ставкой
- Создание и **редактирование** привычки с настройкой ставки (деньги / блокировка)
- «Под риском» за день и модалка «Итог дня» (симуляция списаний/блокировок)
- Сохранение в localStorage

## Планы
