# disbit

Приложение для привычек со **штрафом за пропуск**: не выполнил привычку —
сам заранее выбираешь, что теряешь. Либо **списание денег** (сумму и получателя
выбираешь сам: благотворительность или создателям), либо **блокировка телефона /
выбранных приложений**. Главная идея — не ставки, а **штрафы**: наказание
назначаешь себе сам, заранее.

Дисциплина держится на четырёх опорах:

- **Грейс-день** — пропустил один день: ничего страшного, серия жива.
  Два дня подряд — серия рвётся. Штраф начисляется за каждый пропуск в любом случае.
- **Минимум** — нет сил или мотивации: сделай запасной минимум
  («прочитать 1 страницу») — дисциплина не прервётся.
- **Награды** — сам выбираешь, чем себя поощрить за серию (7 дней → кино).
- **Большие цели** — долгие цели на месяцы, ради которых живут ежедневные привычки.

## Статус

Рабочий прототип. **Ставки симулируются** — реальные списания и блокировки
требуют нативного приложения и платёжной интеграции (это следующие этапы).
ИИ-проверку выполнения задач планируется добавить позже.

## Структура

```
frontend/   — веб-приложение (HTML/CSS/JS, данные в localStorage)
backend/    — сервер Express + SQLite (встроенный node:sqlite)
react/      — React-версия профиля (Vite) — заготовка будущего переезда
```

## Запуск

**Только фронтенд:** открой `frontend/index.html` в браузере —
данные хранятся локально в браузере (localStorage).

**С бэкендом:** запусти сервер (см. [`backend/README.md`](backend/README.md))
и открой http://localhost:3000 — приложение будет синхронизировать данные
с базой SQLite (best-effort: интерфейс работает мгновенно на localStorage,
сервер догоняет).

## Что уже сделано

- **Дизайн v0.4 «deep glass»** — глубокая синяя стеклянная тема (glassmorphism)
  + **8 сменных цветов приложения**: синий, красный, чёрный, светлый (не яркий),
  жёлтый, оранжевый, зелёный, фиолетовый; Unbounded + Manrope (кириллица),
  SVG-иконки, focus-states, reduced-motion, тач-таргеты ≥44px
- **Грейс-день**: один пропуск серию не рвёт, два подряд — рвут
  (штраф начисляется за каждый пропуск независимо от грейса)
- **Минимум**: запасной вариант привычки — кнопка «минимум» держит серию
- **Похвала и награды**: тосты при 100% дня, награды за серии выбираешь сам,
  баннер «награда доступна» на главном экране
- Экран «Сегодня»: полоска недели (тап → шторка дня), кольцо прогресса,
  штраф дня, полоска большой цели, карточки привычек
- **Экран «Календарь»**: месячная сетка, состояния дней, **отметки задним числом**
  (влияют на стрики; зафиксированные штрафы не отменяются)
- **Экран «Профиль»**: аватар (фото с уменьшением до 256px / эмодзи из каталога /
  монограмма), **ползунок мотивации** + цели и желания, **большие цели**
  с прогрессом, **друзья** (локальный список), **привычки на будущее** (бэклог
  с кнопкой «начать»), смена темы, настройки, экспорт/импорт JSON
- **Экран «Статистика»**: недельная диаграмма (% + сравнение с прошлой неделей,
  бары по дням), heatmap за 16 недель, стрики, история штрафов
- **Автоитог дня**: пропуски прошедших дней фиксируются автоматически
  + модалка «Пока тебя не было»
- **Hash-роутинг**: `#today` / `#calendar` / `#stats` / `#profile`
- **Бэкенд на SQLite** (без нативных зависимостей) + синхронизация фронтенда с API
- **react/** — профиль на React (Vite): аватар, мотивация, цели, друзья, темы

## Планы

- Авторизация и привязка привычек к пользователям
- Платежи (Stripe/ЮKassa) — реальное списание ставки
- Нативное приложение (реальная блокировка приложений)
- ИИ-проверка выполнения привычек
- Напоминания / push-уведомления
