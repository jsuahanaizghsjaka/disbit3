# disbit

Приложение для привычек с **добровольной ставкой**: если не выполнил привычку —
ты сам заранее выбираешь, что теряешь. Либо **списание денег** (сумму и получателя
выбираешь сам: благотворительность или создателям), либо **блокировка телефона /
выбранных приложений**. Плюс трекер: дни недели и стрики 🔥.

Идея: помочь тем, кто хочет меняться, но кому не хватает дисциплины и самоконтроля.

## Статус

Рабочий прототип. **Ставки симулируются** — реальные списания и блокировки
требуют нативного приложения и платёжной интеграции (это следующие этапы).
ИИ-проверку выполнения задач планируется добавить позже.

## Структура

```
frontend/   — веб-приложение (HTML/CSS/JS, данные в localStorage)
backend/    — сервер Express + SQLite (встроенный node:sqlite)
```

## Запуск

**Только фронтенд:** открой `frontend/index.html` в браузере —
данные хранятся локально в браузере (localStorage).

**С бэкендом:** запусти сервер (см. [`backend/README.md`](backend/README.md))
и открой http://localhost:3000 — приложение будет синхронизировать данные
с базой SQLite (best-effort: интерфейс работает мгновенно на localStorage,
сервер догоняет).

## Что уже сделано

- **Дизайн-система v0.3**: тёмная тема «cinema dark» с амбер/изумруд-акцентами,
  шрифты Unbounded + Manrope (кириллица), SVG-иконки (Lucide-style) вместо эмодзи,
  focus-states, reduced-motion, тач-таргеты ≥44px
- Экран «Сегодня»: неделя с индикаторами дней (тап → шторка дня), кольцо прогресса,
  карточки привычек, «под риском», модалка «Итог дня»
- **Экран «Календарь»**: месячная сетка с навигацией, состояния дней
  (выполнено/частично/пропуск), **отметки задним числом** (ретро-редактирование
  влияет на стрики; зафиксированные списания не отменяются)
- **Экран «Профиль»**: имя + аватар, сводка, настройки, экспорт/импорт JSON,
  очистка данных
- Создание и **редактирование** привычки: иконка, цвет, цель (отметка/счётчик), ставка
- **Расписание по дням недели** — стрики учитывают расписание (выходные не рвут серию)
- **Экран «Статистика»**: heatmap за 16 недель, сводные показатели, стрики,
  история списаний
- **Автоитог дня**: прошедшие дни закрываются автоматически, пропуски — в журнал
  + модалка «Пока тебя не было»
- **Hash-роутинг**: экраны доступны по ссылкам `#today` / `#calendar` / `#stats` / `#profile`
- **Бэкенд на SQLite** (без нативных зависимостей) + синхронизация фронтенда с API
