## ПРИЛОЖЕНИЕ (Capacitor поверх замороженного веба)

### День 3 — вт 14.07: запускается на телефоне

- Утро: поставить Android Studio + JDK 17 (качается долго — включить первым делом).
- Шрифты Onest/Unbounded локально (woff2 + @font-face) — офлайн обязателен.
- `npm i @capacitor/core @capacitor/cli @capacitor/android` → `npx cap init disbit io.disbit.app --web-dir frontend` → `npx cap add android` → `npx cap run android`.
- Прогон WebView: док/safe-area, шторки, клавиатура, blur-производительность.

**DoD:** disbit открывается как приложение на твоём телефоне.

### День 6 — пт 17.07: сборка и кабинеты

- Keystore: `keytool -genkey` → **два бэкапа** (облако+флешка). Потеря = приложение мертво для обновлений.
- Подписанный AAB + APK, `versionCode 1`, `versionName "1.0.0"`.
- Скриншоты 5–6 (Сегодня, Календарь+цели, Статистика, Медали, тёмная+светлая), баннер 1024×500.
- Тексты: краткое (80 зн.), полное (до 4000), БЕЗ слов «ставки/азарт» — «самодисциплина, символические штрафы (симуляция)».
- RuStore Console: карточка целиком, загрузка сборки. Play Console: internal → closed testing, Data Safety (собираем email), возрастная анкета.

**DoD:** сборки загружены в оба кабинета, карточки заполнены.
