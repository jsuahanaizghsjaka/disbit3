# Картинки достижений

Положи сюда PNG (прозрачный фон, квадрат, 128–256px):

| Файл | Для чего |
|---|---|
| `fire.png` | серии дней + пройденные шаги |
| `dollar.png` | штрафы деньгами |
| `lock.png` | блокировки |
| `trophy.png` | завершённые марафоны |

Пока файла нет — на его месте автоматически показывается SVG-иконка из спрайта,
приложение не ломается. Появился файл — подставится сам, править код не нужно.

Соответствие задаётся в `frontend/script.js`, константа `MEDAL_ART`.

ВАЖНО: архивы с 3dicons содержат исходники `.blend` / `.fbx` — это 3D-модели,
браузер их не отображает. Нужны именно PNG-рендеры (на 3dicons.co есть кнопка
скачивания PNG).
